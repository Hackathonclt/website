HACKATHONclt MMXIII

TECH STACK

HackathonCLTMMXIII is designed to highlight Charlottes Tech Community. As a community event, the overriding requirement for the tech stack is that the tools used to process and present the data be Open Source and accessible to the public at large. The required tech stack is intentionally vague with only a few additional requirements and recommendations as provided below.

THIS IS A WORK IN PROGRESS AND DETAILS MAY CHANGE BEFORE START OF HACKATHON.

REQUIREMENTS:

  1. All required software to process the provided data and any additional data and data access APIs used in the hackathon must be Open Source.
  2. Presentations must be delivered in a web browser.
  3. Algorithms and technology used or developed in the Hackathon will be Open Source and must be published publicly, e.g. on Github.

RECOMMENDATIONS:

  1. Usage of public data feeds and their APIs such as twitter, facebook, and public geo servers is encouraged to augment the provided data and build a good story.
  2. The problem will fundamentally be a big data problem. So scalable data processing such as the MapReduce style processing used by many Hadoop Framework components or other NoSQL data solutions such as MongoDB are preferred.
